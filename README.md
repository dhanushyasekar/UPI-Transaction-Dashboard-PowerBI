## Overview
This project is an interactive Power BI dashboard built to analyze UPI transaction data and identify monthly transaction trends.

## Tools Used
- Power BI
- Microsoft Excel
- DAX
- Data Visualization

## Dataset
The dashboard uses an Excel dataset containing UPI transaction details.

## Dashboard Features

### Transaction and Balance Analysis
- Monthly transaction trends
- Transaction comparison
- Balance analysis
- Interactive filters

### Monthly Transaction Analysis
- Detailed transaction breakdown
- Month-wise comparison
- Dynamic filtering

## Key Insights
- Identified monthly transaction patterns
- Compared transaction volumes across periods
- Analyzed balance trends

## Project Files
- .pbix file contains Power BI dashboard
- Excel file contains source data
